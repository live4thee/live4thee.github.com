Number of Code Commits

今天是加入创业整整三个月。除去清明、五一以及端午加上自己请假，一共休息 了两个礼拜，实际为两个半月。其间 Java 代码相关的改动提交了 46 个， Python相关 16 个，Golang 最多，为 162 次。统计方法也很简单：

$ for r in $repos; do
    cd $r
    git log --since '3 months ago' --author live4thee | grep -c ^commit
    cd - > /dev/null
done
