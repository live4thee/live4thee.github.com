Kubuntu 20.04

Mar 7, 2020 12:31 · 218 words · 1 minute read KDE

没忍住，提前升级到了 20.04.

输入法

升级后自动安装上了 ibus, 切换中文有点问题。删掉 ibus 保留 fcitx 就好。 一个手快，多删了下面两个包，导致使用 gtk 的程序没法切换输入法，重新装 回来。

sudo apt-get install fcitx-frontend-gtk2
sudo apt-get install fcitx-frontend-gtk3

plymouth

碰到 plymouth 出错的问题。

sudo apt purge --reinstall plymouth

thumbnail.so

Thumbnail.so 导致 CPU 负载高 的问题，貌似还没解决。

Python2

Python2 全都删掉了。
